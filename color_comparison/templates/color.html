To use this page, make sure to first select the map you want to inspect more carefully, next select a user who has looked at this map so that his graphs for both the colored, and the grayscale version are generated. This can be done using the drop down menus at the top of the page. The visualizations are generated. For the contour plot only one version can be shown at a time, you can switch between the colored or the grayscale version of the map at the dropdown menu at the top of the contour plot. To view both versions at the same time in a contour plot use the “Manual comparison” page, select the contour plot twice and use the same map and user and pick the different versions.

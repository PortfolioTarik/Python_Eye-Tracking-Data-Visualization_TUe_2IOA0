User Comparison
i To use this page, make sure to first select the map you want to inspect more carefully, next select the version of the map, either grayscale or colored. This can be done using the drop down menus at the top of the page. The selection of different users is done at the top where all users that looked at the selected map are shown, to find a specific user type them into the bar, multiple users can be selected at once (at maximum 3 users are displayed). Keep in mind that not every user has looked at each map, so some combinations may result in empty graphs. The visualizations are generated. For the contour plot only one version can be shown at a time, you can switch between different users using the dropdown menu at the top of the visualization. To view a contour plot for two users at the same time use the “Manual comparison” page and select the contour plot twice with the same map and two different users.
